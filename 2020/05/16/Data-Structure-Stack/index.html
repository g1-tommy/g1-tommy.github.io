[자료구조] 스택

스택의 의미는 “더미, 낟가리”

개요

LIFO(Last-In First-Out) 형태의 자료구조
데이터 삽입이 Bottom -> Up 순서로 이루어지며 삭제는 그 반대의 순서로 이루어짐

Time Complexity

  • Insertion
    • O(1)
  • Deletion
    • O(1)
  • Search
    • O(n)

주요 연산

  • isEmpty(Stack s): 스택의 empty 여부
  • isFull(Stack s): 스택의 full 여부
  • push(Stack s, Element e): 스택에 element 삽입
  • pop(Stack s): top의 element 삭제
  • peek(Stack s): top의 element를 삭제하지 않고 반환
