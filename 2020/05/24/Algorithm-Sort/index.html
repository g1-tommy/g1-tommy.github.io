2020-05-24

정렬 알고리즘의 종류

O(n^2)인 것

  • 버블 정렬 (Bubble Sort)
  • 선택 정렬 (Selection Sort)
  • 삽입 정렬 (Insertion Sort)

O(n*lgn)인 것

  • 병합 정렬 (Merge Sort)
  • 힙 정렬 (Heap Sort)
  • 퀵 정렬 (Quick Sort)

하이브리드 정렬

  • 팀 정렬 (Tim Sort)
  • 인트로 정렬 (Intro Sort)

기타 정렬

  • 기수 정렬 (Radix Sort)
  • 카운팅 정렬 (Counting Sort)
  • 셸 정렬 (Shell’s Sort)
  • 보고 정렬 (Bogo Sort)
  • 보고보고 정렬 (BogoBogo Sort)
  • 대기 정렬 (Sleep Sort)

Bubble Sort

서로 인접한 (adjacent) 두 원소를 검사해서 정렬하는 알고리즘

1회전을 수행하면 가장 큰 자료가 맨 뒤로 이동하기 때문에 (오름차순 가정) 다음 회전시 맨 마지막 데이터는 제외되며, 회차가 늘어날 수록 정렬에서 제외되는 데이터가 증가한다.

Tim Sort

Tim Sort는 Insertion Sort와 Merge Sort를 결합한 알고리즘이다.

근데, Insertion Sort의 시간 복잡도는 O(n^2)…

Insertion Sort는 인접한 메모리와의 비교를 반복하기 때문에 참조 지역성의 원리를 매우 만족한다.
